## Limitations

- This workflow favors high recall at the title-retrieval stage, so raw candidates include many ordinary video/event/spike false positives.
- Every raw title hit was checked against official abstracts/pages before A/B/C promotion, but the result is still not guaranteed exhaustive because full-PDF search was not performed over all papers.
- Abstract-level screening can miss details that only appear deep in the PDF; generated cards remain draft notes and require human verification before citation.
- No systematic code availability checking was performed.
