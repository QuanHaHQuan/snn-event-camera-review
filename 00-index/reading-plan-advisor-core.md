# Advisor Core Reading Plan

Serves the **SECNet ICML 2026 oral to TPAMI extension** direction. It combines the direct method chain with a small set of concrete discussion papers; thematic similarity alone is insufficient.

Current core: **15 papers**.

Composition: **10 required + 5 helpful**. The broader `method_chain`, `discussion`, and `watch` counts in `paper-selection.csv` are candidate-pool labels, not reading assignments.

## Focus Paper

**Scalable Event Cloud Network for Event-based Classification (SECNet), ICML 2026 oral.** This advisor paper defines the extension target and is not counted as a proceedings-corpus entry below.
